# The Mars page

## Installation and running page

1. Clone the page:
    ```bash
    git clone https://github.com/DucZuyVuTM/VuDucZuy_TheMars.git
    ```

2. Run the page, make sure to install Python:
    ```bash
    python app.py
    ```

3. Go to the browser and navigate to http://127.0.0.1:5000/
